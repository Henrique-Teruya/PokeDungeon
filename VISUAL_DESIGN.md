# 🎨 PokeDungeon - Visual Design

## Objetivo Visual

Criar um RPG 2D retrô inspirado em:

- Pokémon Red
- Pokémon FireRed
- Zelda Minish Cap
- Game Boy Advance

O visual deve ser:

- pixel art
- minimalista
- retrô
- simples
- legível
- acadêmico, porém estiloso

---

# 📐 Resolução

Resolução base:
1280x720

Pixel art upscale:
3x ou 4x

Tile size:
32x32

---

# 🎮 Estilo do Jogo

## Overworld

- visão top-down
- movimentação em grid
- mapa pequeno
- poucas salas

## Batalha

- interface estilo Pokémon clássico
- caixa de diálogo
- HP bars
- sprite do jogador e inimigo

---

# 🎨 Paleta de Cores

Inspirada em:

- Pokémon FireRed
- tons suaves
- verde natureza
- marrom dungeon
- azul interface

Evitar:

- neon
- efeitos modernos
- UI futurista

---

# 👤 Player

Características:

- sprite pequeno
- 32x32
- animação simples
- visual retrô

---

# 👾 Pokémon

Sprites:

- estilo GBA
- pixel art simples
- poucos frames

---

# 🧱 Tileset

Necessário:

- chão
- parede
- grama
- porta
- água

---

# 🖼️ HUD

## Dungeon

- HP no canto
- nome do pokémon
- inventário simples

## Batalha

- HP bars
- opções:
  - Attack
  - Item
  - Run

---

# ✨ Efeitos

Manter mínimo:

- transição simples
- fade leve
- sem partículas complexas
